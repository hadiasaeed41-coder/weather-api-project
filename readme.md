# 🌦 Weather API Project

This is my first Python API project.

## Features
- Search weather by city
- Get live temperature
- Get humidity
- Get wind speed

## Technologies Used
- Python
- Requests Library
- Open-Meteo API

## How to Run

1. Install requests:
```
pip install requests
```

2. Run the program:
```
python weatherapp.py
```
